<p><ul><li>By Spatial Mode</li><li>Single Mode</li><li>Multimode</li><li>By Package Type</li><li>N Type</li><li>P Type</li></ul></p>
<p><p>Infrared laser diodes market can be categorized based on spatial mode into single mode and multimode. Single mode diodes emit a single spatial mode beam with high coherence, suitable for precise applications. Multimode diodes emit multiple spatial modes, providing higher power output for broader applications. Additionally, these diodes can also be classified based on package type into N type and P type. N type diodes have negatively doped material, while P type diodes have positively doped material, influencing their electrical properties and applications.</p></p>
